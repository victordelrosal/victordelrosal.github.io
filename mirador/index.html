What it does

Every terminal you have, in one window.

Auto-discovers every claudus session on your Mac, mirrors the conversation, and gives you a single place to read, reply, dictate, attach, search, archive, and route. No login. No cloud. Localhost only.

Auto-discovery

Type claudus in any Terminal tab. Conversation pops into the sidebar within ~1s. Yes, every session.

Push-to-talk voice

Hold space. Speak. Release. Live transcription via the Web Speech API. Edit mid-stream, then talk again. Just like the terminal.

Drop in an image

Drag any image anywhere on the chat. The composer breathes; on send, the path is auto-attached via Claude Code's native @path.

Reply to any bubble

Hover any message, click the chevron, choose Reply, Copy, Speak, or Open in Terminal. WhatsApp-grade ergonomics.

15 themes, 5 typefaces

Royal navy, Tokyo Night, Catppuccin Mocha, Solarized, Sakura, Cielo, Rosa Mexicano, Kluxy, Matrix rainfall, plus more. Each conversation remembers its own.

Editable titles + emojis

Right-click any conversation. Rename. Pick the emoji. Archive. Delete. Auto-derived to start, fully editable forever, persisted per-TID.

Slash autocomplete

Type / and every Claude Code command plus every one of your custom commands surfaces. Arrow keys, Enter to pick.

Archive + Resurrection

Delete a conversation; it vanishes. Type one more message in that terminal and it rises from the dead with sparkles and a witty toast.

Localhost-only. No login.

Binds 127.0.0.1:7321 only. Never internet-facing. Stdlib server, ~2000 lines, no frameworks.

How it works

A small hook. A tmux primitive. A browser.

Mirador adds a Claude Code Stop + UserPromptSubmit hook that walks each transcript and writes a tiny JSON-Lines mirror per session. A 700-line Python stdlib server reads those files and serves a Telegram-style chat UI. Sending types into the real tmux pane.

01

Type claudus

A patched zsh function auto-wraps your shell in a fresh tmux session named tv-<tid>. Claude Code starts inside.

02

Hooks mirror the transcript

On every prompt and every reply, the Claude Code hooks fire and write the conversation as JSON-Lines into ~/.mirador/sessions/.

03

Browser polls + you talk back

The dashboard polls every 800ms. Whatever you type in the browser tmux send-keys's into the real terminal. The terminal is still in charge; you just have a better window on it.

Install

Sixty seconds. No login. No cloud.

One folder, one launcher symlink, one zshrc patch. Localhost. Off-switch always one env var away.

# 1. Clone git clone https://github.com/victordelrosal/mirador.git ~/mirador cd ~/mirador chmod +x bin/mirador hook/mirador-stop-hook.py server.py # 2. Symlink the launcher ln -sf "$PWD/bin/mirador" ~/.local/bin/mirador # 3. Register the hook in ~/.claude/settings.json (see README) # + add the claudus() auto-wrap to ~/.zshrc # 4.
