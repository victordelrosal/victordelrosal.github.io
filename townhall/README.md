# Townhall — Coming-Soon Site

Static landing page for the upcoming book *Townhall: Who Do We Get to Be in the Age of AI* by Victor del Rosal. Self-published, May 2026.

Single-file HTML, no framework. Deep royal indigo palette, dark-first with light toggle. Works at any width down to ~320 px.

## Files

- `index.html` — the page itself (self-contained CSS and JS).
- `og-image.svg` — Open Graph share image (1200×630).
- `README.md` — this file.

## Deploy

Pushed to a private GitHub repo and served via GitHub Pages from the main branch root.

## Update

Edit `index.html` directly and push to main. GitHub Pages picks up the change in a couple of minutes.
