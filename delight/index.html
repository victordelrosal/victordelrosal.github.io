DELIGHT.md

a recursive protocol for human kindness
The One-Liner
while alive; do cat DELIGHT.md | human ; done
The Prompt
NOTICE someone in your world today.
ASK what would make their day unexpectedly better?
DO that thing. However small.
FEEL what happens inside you when you do.
PASS this file to one person.
Acceptance Criteria
  • One specific person (not "people")
  • One specific act (not "be nicer")
  • You did it (not "planned to")
  • You felt something (the hook that runs the loop again)
  • One other person now has DELIGHT.md
The Algorithm 
def delight(human):
    someone = human.notice()         # Encourage: see them
    need = someone.real_need()        # Enlighten: understand, don't assume
    human.act(need)                   # Empower: do the thing
    human.feel()                      # Equip: the feeling is the tool
    someone.receive(DELIGHT_MD)       # Envision: they see it, they want it

# The loop
while True:
    delight(you)
    # that's it. there is no step 2.
The Recursive Proof 
delight(A):
    A delights B
    B.receive(DELIGHT.md)
    delight(B):
        B delights C
        C.receive(DELIGHT.md)
        delight(C):
            ...

No base case. It never terminates. That's the point.
