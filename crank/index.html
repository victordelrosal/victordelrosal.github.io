THE MECHANISM

Strip the mystique and every agent loop is four parts.

The prompt is the cheap part now. The art is almost entirely in parts three and four.

01 · GENERATOR

The next prompt

A template, a task list, or a mutation of the last attempt. In Crank, the director writes its own brief and binary criteria before spending a token.

02 · AGENT RUN

The work

A fleet of specialists, one per independent workstream, each opening with a three-line readback so misalignment dies at line three, not in the artifact.

03 · EVALUATOR

The cold verifier

A separate agent, fresh context every round, sees only the criteria and the artifacts, and is instructed to fail the work. For anything runnable, it drives the artifact like a user.

04 · CONTROLLER

The decision

Investigate why a criterion failed, then iterate structurally, fork rival candidates when stuck, or stop honestly. The verifier’s verdict gates the stop, not the builder’s feelings.

“A loop with a weak evaluator fails confidently a thousand times and bills you for it.” THE WHOLE SKILL IN ONE SENTENCE

THE TRANSMISSION

You stopped being the loop. Now stop working the gearbox.

Switching models by hand gets old fast: Fable 5 to plan, Opus to build, Sonnet for the menial parts, then back again every few minutes. Crank shifts for you. Every agent the loop dispatches carries an explicit model choice, picked by the shape of the task, so you never /model-switch mid-run.

TOP · FABLE 5

Plan and judge

The framing on a high-stakes run, the hardest structural bets, and the final cold verifier. The one seat where the sharpest model earns its keep.

DRIVE · OPUS

Build

The fleet of builders and most per-round verification. The default gear; when in doubt, drive.

LOW · SONNET

The menial passes

Scrapes, grep sweeps, format fixes, bulk plumbing. Cheap work routed to a cheap seat, run at low effort.

An automatic you cannot see is just a black box. So the run opens by telling you which gear it is in, before anything else, and calls out every shift as it happens. You always know whether the work of the moment is Fable, Opus or Sonnet, without asking.

⚙ GEARBOX (automatic). Session gear: Fable (top). Routing this run: planning + final verify stay in top gear (~10%); builders dispatch in drive (Opus); mechanical sweeps in low (Sonnet). I'll announce every shift. ⚙ shifting to low (sonnet): link sweep across 40 pages ⚙ top gear (fable): cold verify, final round

Top gear plans and judges; it never executes. Fable 5 is the hungry model, so it is spent where a single call decides the run and nowhere else: roughly one part in ten. Two honest limits. The loop cannot change the model you are on, only the agents it sends out; when the director itself needs the top gear, it spawns a Fable 5 agent for that one decision and folds the answer back. And a gear is not a promise of quality: it is where the attention was spent, which is exactly why Crank shows you the number instead of asking you to trust it.

The gearbox was always there. Crank just stopped making you work it.AUTOMATIC, NOT STICK

THE DISCIPLINE

Every pin on the cylinder is a rule that earned its place.

Six rules, each traced to a named, fetch-verified source. Hover a pin to find its rule.

The verifier gates the stop

The loop cannot declare done while any binary criterion fails. Self-grading is systematically lenient; the grader lives in a separate context window.

Anthropic loop experiments · CMA Outcomes

The evaluation surface is frozen

Criteria, check scripts and verifier instructions are part of the contract. A fix that edits the test instead of the work is a downgrade in disguise.

Karpathy autoresearch · Anthropic harness law

Ground before you frame

Locate the real artifacts and write an explain-back, every claim marked VERIFIED or ASSUMED, before the contract freezes. The expensive bugs are shared misunderstanding.

Uriostegui · ClineFlow

Baseline before you improve

Measure the unmodified starting state first. Every claim of improvement is proved against the baseline, not against memory of it.

Karpathy autoresearch

Memory that compounds

Verified rules distill to LEARNINGS; reusable scripts promote to a tools library. Run N+1 starts from competence, not from scratch.

Lance Martin · Voyager · AWM

Freedom stops at the gate

Money, reach, publishing, deletion: anything irreversible pauses for a human, or parks in an approval queue when running unattended. The gate never lowers because no one is watching.

Crank doctrine · scheduled mode

THE LINEAGE

Not year zero. A family with a ladder.

ReAct · 2022Reason, act, observe, repeat. One model, one loop, a human watching.
AutoGPT · 2023A goal, and permission to self-prompt. Became famous for spinning forever doing nothing.
Ralph loop · 2025The same prompt file, piped in over and over. Its real move was resetting context to fixed anchors each iteration.
/goal, /loop · 2026The ralph loop, productised until a validator confirms done.
autoresearch · 2026The loop pointed at model training: mutate, run under a fixed budget, keep only what beats a frozen metric.
Crank · nowThe closed-loop, budget-disciplined member of the family: a human designs the path, every step has an eval, and there is a defined point where it stops or hands back.
